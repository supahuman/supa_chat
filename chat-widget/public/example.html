Features

  • Easy one-line integration
  • Customizable themes and colors
  • Responsive design
  • Real-time messaging
  • Session management
  • Multiple client support
  • Framework agnostic
  • Lightweight (~15KB)

Integration Code

Add this code to your website:

<script> window.SupaChatbotConfig = { clientId: 'your-client-id', apiUrl: 'https://your-api-domain.com', theme: 'light', position: 'bottom-right', primaryColor: '#3B82F6', showBranding: true }; </script> <script src="https://your-cdn-domain.com/supa-chatbot.js" async></script>

Custom Styling Example

Advanced configuration with custom styling:

<script> window.SupaChatbotConfig = { clientId: 'your-client-id', apiUrl: 'https://your-api-domain.com', theme: 'dark', position: 'bottom-left', primaryColor: '#FF6B6B', secondaryColor: '#4ECDC4', borderRadius: '16px', fontFamily: 'Inter, sans-serif', showBranding: false, customCSS: ` .supa-chatbot-widget { box-shadow: 0 20px 40px rgba(0,0,0,0.3); } .supa-chatbot-header { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); } ` }; </script>
